# モックアップから実装指示を作成

## 用途
Claude.ai（チャット）で作ったモックアップHTMLを元に、実装指示を生成する

## 手順
1. 指定されたモックアップHTMLファイルを読む
2. 現在のflow-appのコードと比較して差分を特定
3. 以下の形式で実装指示を生成:
   - 変更するファイル名
   - 追加するHTML要素
   - 追加/変更するCSS（具体的なプロパティ値まで）
   - 追加/変更するJavaScript（関数名・ロジック）
   - データソース（Supabaseテーブル名・クエリ）
4. UI設計ルール（CLAUDE.md記載）に準拠しているか確認
5. 実装を実行
6. git commit & push
